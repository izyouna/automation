# .roo Directory

This directory contains configuration and monitoring files for the microservices project, organized into four categories:

## Subdirectories

### heartbeat/
Contains health check configurations, heartbeat scripts, and monitoring endpoints for service availability.

### hitl/
Human-In-The-Loop (HITL) configurations, approval workflows, and manual intervention scripts.

### compliance/
Compliance check configurations, regulatory requirements, audit logs, and policy enforcement.

### governance/
Governance policies, access control, decision frameworks, and operational guidelines.

## Usage

Each subdirectory may contain:
- Configuration files (YAML/JSON)
- Scripts (Python, PowerShell, Bash)
- Documentation
- Templates

Place relevant files in the appropriate subdirectory to maintain organization.
